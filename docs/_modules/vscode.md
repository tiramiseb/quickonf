---
title: VS Code
---

| Instruction          | Action                       | Arguments          |
| -------------------- | ---------------------------- | ------------------ |
| `vscode-extension`   | Install VS Code extensions   | List of extensions |
| `vscodium-extension` | Install VS Codium extensions | List of extensions |

Example:

```yaml
- Go extension for VSCode:
    - vscode-extension:
        - golang.go
```
